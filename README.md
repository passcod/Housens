# Housens

Project to make my own low-power house sensors.

Current iteration (r002):

- Bluetooth
- ambient light (OPT3005)
- pressure, humidity, temperature (BME280)
- carbon dioxide (STCC4)
- STM32WG core
- powered from CR2032
- 50x50mm
